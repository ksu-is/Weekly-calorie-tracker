# Weekly Calorie Tracker

## Description
This project is a Python application that allows users to track daily calorie intake and view a weekly summary.

## Features
- Input daily calorie intake
- Store data in a file
- Calculate weekly total and average
- Identify highest and lowest calorie days
- Compare intake to a calorie goal

## Target Users
Individuals who want a simple way to track calorie intake without using complex apps.

## Technologies
- Python
- File handling
- Optional Flask interface
